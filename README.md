# MathModelingCompetition

Reference materials for mathematical modeling competition preparation.

## Contents

- **Agent output examples** (`开发可用文件/agent成品合集/`) — Model solutions generated by ChatGPT, Fable (Claude), Gemini 3.1 Pro, and DeepSeek, illustrating how different LLMs approach competition problems.
- **Award-winning papers** (`开发可用文件/获奖论文合集/`) — Past winning entries for reference on structure, methodology, and presentation.
- **Problem statements & data** (`开发可用文件/题目/`) — Competition problem (Problem A) with attachments and the 2025 formatting guide.

## Purpose

This repository serves as a personal workspace for learning and preparing for mathematical modeling competitions. It is not intended to be a general-purpose library.
